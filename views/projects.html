Randsom

Randsom is an app that takes a phrase and matches the letters in each word to an image of that letter from the Pinterest API and then scrambles and displays the phrase. Helpful if you are playing a prank on someone and don't want to waste money on glue and magazine cutouts. If you are actually ransoming something, please don't use our app. We recommend you return whatever you are ransoming immediately
