Marquee

Marquee is a concert sharing app that streamlines coordinating attendance to concerts and events among groups of friends. Users can search the Ticketmaster API for upcoming concerts, form groups with other users, and create shared lists where member can vote on which concerts to attend

Technologies Used:

  • AngularJS
  • Express
  • MongoDB/Mongoose
  • Node.js

Randsom

Randsom is an app that takes a phrase and matches the letters in each word to an image of that letter from the Pinterest API and then scrambles and displays the phrase. Helpful if you are playing a prank on someone and don't want to waste time and money on glue and magazine cutouts. If you are actually ransoming something, please don't use our app. We recommend you return whatever you are ransoming immediately

Technologies Used:

  • AngularJS
  • Ruby on Rails

Rails Recipes

Rails Recipes is a way to completely optimize your meal planning experience. Users can make make a weekly recipe list where they can view their schedule, add recipes to certain days of the week, and view a shopping list of all the ingredients they need

Technologies Used:

  • Ruby on Rails
